Conceitos Fundamentais: Variáveis

1. O que é uma variável?

Uma variável é um espaço na memória do computador onde você guarda informações que seu programa precisa usar.

Analogia:

Pense em uma caixa de armazenagem:

  • A caixa tem um rótulo (nome da variável)
  • Dentro dela você coloca um tipo de coisa (o tipo de dado)
  • E guarda um valor específico

Exemplo Visual:

  • nome_paciente: Nome da variável (rótulo)
  • "João Silva": Valor armazenado
  • Texto (str): Tipo de dado

Como Nomear Variáveis

Convenção: snake_case

✔ NOMES CORRETOS (Use este padrão!)

Bom: claro, descritivo, tudo minúsculo com underscore.

nome_paciente = "João Silva"
idade_paciente = 45
tem_convenio = True
valor_consulta = 150.50

Aceitável (uma palavra): nome = "Maria", idade = 30

Nomes Errados (Não Use!)

X Ruim: não descreve o que é

x = "João" | n = 45 | pac = "Maria"

X Ruim: usa camelCase (Java usa, Python não)

nomePaciente = "João" | idadePaciente = 45

X Ruim: tem espaços ou caracteres especiais

nome paciente = "João" (Erro: espaço)
idade-paciente = 45 (Erro: hífen)

Regras de Ouro para Nomes

  1. Sempre em minúscula (exceto constantes).
  2. Separe palavras com underscore (_).
  3. Seja descritivo: diga o que a variável armazena.
  4. Evite o óbvio ruim: a, x, dados não explicam nada.
  5. Evite caracteres especiais: apenas letras, números e underscore.

Tipos de Dados Primitivos: STRING

A. STRING (str) - Texto

Armazena texto (sequência de caracteres).

nome_cliente = "Ana Silva"
email = "ana@email.com"
cpf = "123.456.789-00"
endereco = "Rua das Flores, 42, Joinville - SC"
# Exibir tipo: print(type(nome_cliente)) -> <class 'str'>

Quando usar: Nomes, emails, endereços, identificadores que não serão calculados, qualquer informação em forma de texto.

Tipos de Dados Primitivos: INTEGER

B. INTEGER (int) - Número Inteiro (Sem Decimais)

Armazena números sem casa decimal.

idade = 30
quantidade_itens = 5
numero_pedido = 12345
ano_nascimento = 1993
# Exibir tipo: print(type(idade)) -> <class 'int'>

Quando usar: Idades, quantidades, contagens, IDs, números de pedidos (que não usam decimais), qualquer contagem inteira.

Tipos de Dados Primitivos: FLOAT

C. FLOAT (float) - Número com Decimais

Armazena números com casa decimal.

valor_consulta = 150.50
preco_produto = 29.99
taxa_juros = 0.05
altura_metro = 1.75
# Exibir tipo: print(type(valor_consulta)) -> <class 'float'>

Quando usar: Preços, valores monetários, proporções, percentuais, medidas (altura, peso, distância).

Tipos de Dados Primitivos: BOOLEAN

D. BOOLEAN (bool) - Verdadeiro ou Falso

Armazena apenas True ou False.

tem_convenio = True
pedido_entregue = False
cliente_ativo = True
produto_disponivel = True
# Exibir tipo: print(type(tem_convenio)) -> <class 'bool'>

Quando usar: Status de ativação (ativo/inativo), confirmações (pago/não pago, entregue/não entregue), qualquer condição que seja sim/não.

Tabela de Referência Rápida

Tipo Nome Exemplo Quando Usar

int Integer (Inteiro) 30, 5, 12345 Idades, quantidades, IDs
float Float (Decimal) 150.50, 0.05, 1.75 Preços, proporções, medidas

Formatando Strings com f-strings

O que é uma f-string?

Uma f-string (formatted string literal) permite misturar texto com variáveis de forma legível.

Sintaxe Básica:

Use 'f' antes das aspas e coloque a variável entre chaves {}.

mensagem = f"Texto aqui {variavel} mais texto"

Exemplo 1: Relatório Simples

Código:

nome_paciente = "João Silva"
idade_paciente = 45
tem_convenio = True
valor_consulta = 150.50

relatorio = f"Paciente: {nome_paciente}, Idade: {idade_paciente} anos, Convênio: {tem_convenio}, Valor: R$ {valor_consulta}"

print(relatorio)

Resultado:

Paciente: João Silva, Idade: 45 anos, Convênio: True, Valor: R$ 150.5

Exemplo 2: Estrutura Mais Legível

Código (usando quebras de linha):

relatorio = f"""
FICHA DO PACIENTE
Nome: {nome_paciente}
Idade: {idade_paciente} anos
Convênio: {tem_convenio}
Valor Consulta: R$ {valor_consulta}
"""
print(relatorio)

Resultado:

FICHA DO PACIENTE
Nome: João Silva
Idade: 45 anos
Convênio: True
Valor Consulta: R$ 150.5

Exemplo 3: Sistema de Delivery

nome_cliente = "Maria"
quantidade_itens = 5
valor_total = 89.90
pedido_entregue = False

mensagem = f"""
Olá {nome_cliente}!
Seu pedido com {quantidade_itens} itens
Total: R$ {valor_total}
Status: {'Entregue' if pedido_entregue else 'Em Preparação'}
"""
print(mensagem)

Resultado:
Olá Maria!
Seu pedido com 5 itens
Total: R$ 89.9
Status: Em Preparação

Erros Comuns: Tipo de Dado

Erro 1: Confundir string com número

X ERRADO

idade = "30" # É texto, não número!
print(idade + 5)
# Erro! Não pode somar texto com número

✔ CORRETO

idade = 30 # É número inteiro
print(idade + 5)
# Resultado: 35

Erros Comuns: Sintaxe no Nome

Erro 2: Usar espaços ou caracteres especiais

X ERRADO

nome paciente = "João" # Erro de sintaxe (espaço)
idade-paciente = 45    # Erro de sintaxe (hífen)
@valor = 100           # Erro de sintaxe (@)

✔ CORRETO

nome_paciente = "João"
idade_paciente = 45
valor = 100

Erros Comuns: Esquecer a f-string

Erro 3: Sintaxe da string formatada

X ERRADO (sem 'f')

nome = "Maria"
mensagem = "Olá {nome}"
# Imprime a string literal, não o valor
print(mensagem)
# Resultado: Olá {nome}

✔ CORRETO (com 'f')

nome = "Maria"
mensagem = f"Olá {nome}"
# Interpreta a variável
print(mensagem)
# Resultado: Olá Maria

Dúvidas Frequentes

P: Por que não posso usar um espaço no nome da variável?
R: Python interpreta espaço como separador de comandos. Use underscore (_) em seu lugar.

P: Qual a diferença entre 30 (int) e "30" (str)?
R: 30 é um número que você pode calcular; "30" é texto que parece um número.

P: Posso mudar o tipo de uma variável depois?
R: Sim! (ex: x=10, depois x="texto"), mas não se recomenda fazer isso no mesmo programa.

P: f-string é obrigatória?
R: Não, mas é a forma mais moderna, legível e melhor prática desde Python 3.6+.
